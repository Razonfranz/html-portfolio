## ğŸ”§ Customization

### Update Personal Information
Edit `index.html`:
- Hero section: Update name and title
- About section: Update bio
- Projects: Replace with your own work

### Change Colors
Edit `css/styles.css`:
```css
:root {
  --primary-color: #2563eb;      /* Main brand color */
  --primary-dark: #1d4ed8;       /* Darker variant */
  --text-color: #1f2937;         /* Main text color */
}
```

### Update Contact Form
Replace Formspree form ID in `index.html`:
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```

Get your free form ID at [Formspree.io](https://formspree.io/)
